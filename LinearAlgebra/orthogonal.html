2. Gram-Schmidt正交化

线性无关向量组未必是正交向量组，但正交向量组又是重要的，因此现在就有一个问题：能否

  • 从一个线性无关向量组a1,a2,⋯,ama_1, a_2, \cdots, a_ma1​,a2​,⋯,am​出发，
  • 构造出一个标准正交向量组e1,e2,⋯,eme_1, e_2, \cdots, e_me1​,e2​,⋯,em​，
  • 并且使向量组a1,a2,⋯,ara_1, a_2, \cdots, a_ra1​,a2​,⋯,ar​与向量组e1,e2,⋯,ere_1, e_2, \cdots, e_re1​,e2​,⋯,er​等价r=(1,2,...,m)呢?

答：施密特正交化方法。
以3个向量组成的线性无关组为例

2.1. 第一步：线性无关向量组（a1,a2,a3） ---> 正交向量组(b1,b2,b3)

令: b1 = a1
b2 = a2 - k b1
b3 = a3 - k1 b1 - k2 * b2

由于b1、b2、b3互相正交，
b1 b2 = 0 ==> k = <a2,b1><b1,b1>\frac{<a2, b1>}{<b1, b1>}<b1,b1><a2,b1>​
b1 b3 = 0 && b2 * b3 = 0 ==> k1=<a3,b1>b1,b2k1 = \frac{<a3, b1>}{b1, b2}k1=b1,b2<a3,b1>​， k2=<a3,b2>b2,b2k2 = \frac{<a3, b2>}{b2, b2}k2=b2,b2<a3,b2>​
